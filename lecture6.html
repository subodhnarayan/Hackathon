Lecture 6: Deploy Your First Project | Media Tag | Multi Page Website

Lecture Video

Key Concepts from this Lecture

  • HTML Media Tags: HTML5 introduced native tags for media, eliminating the need for plugins like Flash. The primary tags are `<video>` for visual media and `<audio>` for sound.
  • Essential Media Attributes: To make media players functional, we use boolean attributes like `controls` (to show the player interface), `autoplay` (to start playing on load, usually requires `muted`), and `loop` (to repeat the media).
  • Multi-Page Websites: A real website isn't just one page. It's a collection of linked HTML files (e.g., `index.html`, `about.html`, `contact.html`). Consistent navigation links on each page allow users to move between them seamlessly.
  • Deployment: This is the process of taking your website's files from your local computer and putting them on a web server so that anyone in the world can access them via a public URL. Services like Netlify, Vercel, or GitHub Pages make this process very simple.

Detailed Notes & Examples

1. HTML Media Tags

The `<video>` and `<audio>` tags allow you to embed media directly into your webpage without any special plugins.

Example A: Embedding a Video

This example shows a video with controls. The `<source>` tag is the professional way to provide a video file, ensuring browser compatibility.

<!-- From video.html -->
<h1>This is our Video page</h1>

<video autoplay controls height="400px">
    <source src="./rohit.mp4" type="video/mp4">
</video>

Example B: Embedding an Audio File

Similarly, the `<audio>` tag embeds a sound file with a player interface when the `controls` attribute is present.

<!-- From audio.html -->
<h1>This is our Audio page</h1>
<audio controls>
    <source src="./wavy.mp3" type="audio/mp3">
</audio>

2. Creating a Multi-Page Website

A multi-page website uses a consistent header with navigation links on every page. Each link points to another HTML file in your project, creating a cohesive user experience. The example below shows the standard navigation header used across all your pages.

<!-- Consistent header from all pages -->
<header>
    <h1>This is our Home Page</h1>
    <ul>
        <li><a href="index.html">HomePage</a></li>
        <li><a href="contact.html">Contact Us</a></li>
        <li><a href="course.html">Courses</a></li>
        <li><a href="about.html">About us</a></li>
    </ul>
</header>

<main>
    <h2>This is our course selling website</h2>
    <img src="courseimagelink" height="500px" alt="course">
</main>

<footer>
    <p>Copyright © All rights are reserved</p>
</footer>
